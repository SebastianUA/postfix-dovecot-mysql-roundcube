## postfix-dovecot-mysql-roundcube
postfix + dovecot with mysql and roundcube


##INSTALL

Install git.

CentOS/RedHat/Fedora:
`# yum install git`

Debian/ubuntu/Mint:
`# apt-get install git`

`# cd /usr/local/src/ && git clone https://github.com/SebastianUA/postfix-dovecot-mysql-roundcube.git`
